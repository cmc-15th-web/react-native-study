## week2 과제

### 1. 웹 페이지 생성 (완료)

App 프로젝트 뿐만 아니라 Web 프로젝트를 생성합니다. (React, Next.JS 등 사용)

### 2. 지도 불러오기

- Web 페이지 상에서 지도를 띄워줍니다. (완료)
- WebView를 사용하여 App 내에서 해당 Web 페이지를 불러옵니다. (react-native-webview 라이브러리 활용)
- 지도는 카카오, 네이버, 구글 중 하나의 지도를 임의로 선정해서 구현해주세요. (완료-네이버)

### 3. 지도 상호 작용 (완료)

- 사용자가 지도를 확대, 축소할 수 있어야 합니다.
- 지도를 드래그하여 이동할 수 있어야 합니다.

### 4. 현재 위치 표시

- react-native-permissions를 사용하여 위치권한을 받아옵니다.
- React Native와 WebView 간에 현위치를 주고받기 위해 Bridge를 구현합니다.
- 기기의 현재 위치 정보를 React Native에서 받아와서 WebView 내에서 현 위치의 핀을 표시합니다.

### 5.(선택) 즐겨찾기 기능

- React Native와 WebView 간에 즐겨찾기 목록를 주고받기 위해 Bridge를 구현합니다.
- WebView 내에서 마커를 클릭하고, 비활성화된 즐겨찾기 버튼을 누르면 해당 위치의 정보를 React Native로 전달하고, React Native에서 즐겨찾기 목록에 추가합니다.
- 가능하다면 주소정보도 같이 저장해주세요.
